# opabrow Todo

opabrow 的方向是“轻巧、透明、适合工作间隙使用的浮层浏览器”。优先补齐单页浏览体验，不急于变成完整的多标签 Chrome 替代品。

## 已完成

- [x] 本地历史记录：地址栏建议、原生菜单最近 10 条、`Cmd+Shift+Y` 搜索/重新打开/清空
- [x] 收藏夹与 Chrome、Safari、HTML 书签导入
- [x] Chrome 密码 CSV 导入与手动填充
- [x] 主页自定义、手机模式、透明度与窗口置顶
- [x] Bilibili 视频页自动切换网页全屏

## 下一步

### P0 - 浏览器基本功

- [ ] 页面内查找：`Cmd+F`、上一个/下一个匹配项、匹配数量和关闭按钮
- [ ] 会话恢复：重启后恢复最后页面、窗口位置和尺寸、透明度、置顶及手机模式状态
- [ ] 下载管理：显示下载进度、完成/失败状态，并可在 Finder 中定位文件

### P1 - 隐私与可控性

- [ ] 网站权限中心：摄像头、麦克风、通知和定位的允许/拒绝控制
- [ ] 网站数据管理：按站点清理 Cookie、缓存和本地存储
- [ ] 历史记录单条删除与按日期分组

### P2 - 贴合浮层场景的差异化

- [ ] 独立资料夹：工作与休闲分开保存 Cookie、登录态、收藏和历史
- [ ] 快捷窗口预设：一键切换专注、视频和手机网页的尺寸与透明度
- [ ] 可选的快捷键说明页，便于第一次使用时发现核心操作

## 暂不优先

- [ ] 多标签栏：会削弱单页浮层的轻量定位，等独立资料夹和会话恢复成熟后再评估
- [ ] 扩展商店：安全边界和兼容成本很高
- [ ] 云同步：在本地优先和隐私方案明确前不引入账号体系
